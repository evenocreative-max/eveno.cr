Mobile Test untuk EVENO CREATIVE LAB

Simulasi tampilan mobile di bawah ini menunjukkan bagaimana website akan terlihat di perangkat mobile. Teks "EVENO CREATIVE LAB" seharusnya sekarang terlihat jelas di bagian atas.

Perbaikan yang dilakukan:

  • Memperbaiki struktur HTML yang tidak valid (menghapus h2 di dalam h1)
  • Menambahkan padding-top pada hero section untuk mobile
  • Menyesuaikan ukuran font untuk berbagai ukuran layar
  • Memastikan text alignment yang proper untuk mobile
